# tim's thoughts

Developer blog built with [Hugo](https://gohugo.io/) and [shibui](https://github.com/ntk148v/shibui) theme.

## Quick Start

```bash
# clone with theme submodules
git clone --recurse-submodules https://github.com/timothyckl/blog
cd blog

# run dev server
hugo server -D
# visit http://localhost:1313
```

## File Structure 

```bash
.
├── content/        # posts & pages
├── themes/         # theme submodules
├── hugo.toml       # site config
└── README.md
```

## Writing

```bash
hugo new posts/my-first-post.md
```
